## Architecture Decisions

### Persistence Strategy
Both features use `SharedPreferences` with JSON serialization (same pattern as `ThemeModeNotifier`):
- Avatar: key `avatar_config`, full JSON with enum names
- Biometrics: key `biometric_config`, JSON with numeric values

This avoids Drift schema migrations for simple user preferences while keeping the data offline-first and instant.

### Rendering Approach
The avatar uses `CustomPainter` with a layered draw order (back to front):
1. Body/torso (outfit colored, or skin if shirtless)
2. Outfit details (neckline, sleeves, pockets)
3. Arms (skin-colored rectangles)
4. Head (large rounded square — the signature Mii blockiness)
5. Hair (overlay shapes per style)
6. Expression (eyes + mouth + optional eyebrows)
7. Head accessory (headband/sweatband over hair)

Every shape gets `PaintingStyle.fill` + 2px `PaintingStyle.stroke` black border.

### Bottom Sheet Pattern
The biometric editor avoids keyboard-triggered overlap by using exclusively button/slider input:
- 64px tall `GestureDetector` buttons with 2px black borders for +/- increment
- `Slider` for rapid value scrubbing across the full range
- `StatefulBuilder` for local editing state, provider setter called on Save
- Same `showModalBottomSheet` + `DraggableScrollableSheet` pattern as existing dialogs

### Responsive Layout
`BiometricCard` uses `LayoutBuilder` to switch between:
- **Horizontal** (3 equal columns) when width ≥ 300px
- **Vertical** (1 full-width + 2 half-width) when width < 300px

## Key Code Patterns

### Provider + SharedPreferences (reused across both features)
```dart
final avatarProvider = StateNotifierProvider<AvatarNotifier, AvatarConfig>(
  (ref) => AvatarNotifier(),
);

class AvatarNotifier extends StateNotifier<AvatarConfig> {
  AvatarNotifier() : super(const AvatarConfig()) { _load(); }

  Future<void> _load() async {
    final prefs = await SharedPreferences.getInstance();
    final jsonStr = prefs.getString(_kAvatarKey);
    if (jsonStr != null) {
      state = AvatarConfig.fromJson(json.decode(jsonStr));
    }
  }

  Future<void> _save() async {
    final prefs = await SharedPreferences.getInstance();
    await prefs.setString(_kAvatarKey, json.encode(state.toJson()));
  }
}
```

### CustomPainter Component Layering
```dart
@override
void paint(Canvas canvas, Size size) {
  _drawBody(canvas, w, h, center);
  _drawArms(canvas, w, h, center);
```
