## Data Architecture

All user profile data lives in a single `BiometricConfig` object persisted via SharedPreferences:

```json
{
  "fullName": "string",
  "age": int,
  "currentWeight": double,
  "targetWeight": double,
  "height": double
}
```

BMI is computed on-the-fly: `(weight_lbs × 0.453592) / (height_inches × 0.0254)²`

Avatar config remains a separate SharedPreferences entry (`avatar_config`).
