## Architecture

### Theme Provider

```dart
// lib/providers/theme_provider.dart
final themeModeProvider = StateNotifierProvider<ThemeModeNotifier, ThemeMode>(
  (ref) => ThemeModeNotifier(),
);
```

- Persists via SharedPreferences key `'theme_mode'`
- Default: `ThemeMode.system`
- Values: `'system'` | `'light'` | `'dark'`

## Key Design Decisions

1. **Kawaii aesthetic preserved in dark mode** - Deep purple-black backgrounds (`#0F0F1A`) instead of pure black, maintaining the neon pink (`#FF6B9D`) and electric blue (`#4ECAFF`) accents.

2. **`Theme.of(context)` for surface colors** - Inline `Container` decorations now use `Theme.of(context).colorScheme.surface` instead of `GritTheme.surface` to respond to theme changes.

3. **Brightness check for plate labels** - Plate calculator uses `Theme.of(context).brightness == Brightness.dark` for text on colored plate backgrounds, since plate colors are real-world barbell colors that don't change with theme.

4. **SharedPreferences for persistence** - Simple string serialization via `.name` property, compatible with future theme options.
