## Key Implementation Details

### Generator Algorithm

1. **Fetch exercises** filtered by equipment mapping
2. **Resolve template framework** based on days (2=FullBody, 3=PPL/Strength, 4=Upper/Lower, 5=PHAT)
3. **Fill slots** per day: define muscle group slots, query available exercises matching muscle+equipment, pick random non-duplicates
4. **Apply focus reordering**: move targeted muscles to front of day sequence; at Intermediate level, swap 1 minor accessory for extra focused exercise
5. **Generate dynamic name** and assign day names

### Re-roll Logic

`rerollExercise()` finds exercises with same `targetMuscle` and compatible equipment, excluding already-used IDs, and returns a random replacement.

### Dynamic Naming

- Beginner + focus → "Beginner {Focus} Crusher (Split)"
- Intermediate + no focus → "Elite Power Builder (Cycle)"
- Moderate + focus → "Power {Focus} Shaper (Split)"
